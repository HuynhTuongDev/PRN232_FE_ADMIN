# PRN232 FE Admin

Admin dashboard frontend for the PRN232 GoRide project. Built with Next.js 14.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server (port 3003)
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_API_URL=https://prn-232-be.vercel.app/api/v1
```

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: CSS Modules / Global CSS
